# Darija Translator - Chrome Extension - Hanane Malki

## Description

Chrome extension for quick text translation to Moroccan Darija and other languages. Translate selected text on any webpage with a simple right-click context menu.

## Features

- **Context Menu Translation**: Right-click any selected text to translate
- **Multiple Languages**: Darija, French, Spanish, German, Arabic, English
- **Quick Access**: Popup interface for instant translation
- **Clean UI**: Simple and intuitive design
- **Secure**: Uses your backend API for translations


## Installation

### Load Extension in Chrome

1. **Download/Clone the extension folder**
```bash
git clone [REPO_URL]
cd chrome-extension
```


2. **Load in Chrome**
   - Open Chrome and go to `chrome://extensions/`
   - Enable **Developer mode** (toggle in top-right)
   - Click **Load unpacked**
   - Select the `chrome-extension` folder
   - Extension icon appears in toolbar

3. **Pin the extension** (optional)
   - Click the puzzle icon in Chrome toolbar
   - Pin the Darija Translator extension

## Usage

### Method 1: Context Menu (Right-Click)

1. Select any text on a webpage
2. Right-click the selection
3. Choose **"Translate to [Language]"** from context menu
4. View translation in notification or popup

### Method 2: Popup Interface

1. Click the extension icon in toolbar
2. Paste or type text in the text area
3. Select target language
4. Click **Translate** button
5. View result instantly
